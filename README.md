# bigdata-log-analysis
Bu proje, büyük ölçekli ham web sunucusu log verisini etkin bir şekilde işlemek, analiz etmek ve görselleştirmek için uçtan uca, dağıtık bir veri işleme pipeline'ı (borusu) kurmayı amaçlamıştır. Bu çalışma, modern Big Data teknolojilerini kullanarak verimlilik, depolama optimizasyonu ve düşük gecikmeli erişim konularına odaklanmaktadır.
